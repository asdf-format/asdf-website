.. _asdf-python:

ASDF for Python
===============

The :ref:`asdf <asdf:asdf>` python implementation is available on `github <https://github.com/asdf-format/asdf>`__ and `pypi <https://pypi.org/project/asdf/>`__. Python users can install the package using pip:

.. code-block:: bash
    
    $ pip install asdf
