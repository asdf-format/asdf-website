ASDF for C (libasdf)
====================

`libasdf <https://github.com/asdf-format/libasdf>`__ is an officially supported prototype implementation of the ASDF Standard in C. Although not yet feature-complete, the library supports ASDF's core capabilities of reading metadata from YAML and reading from n-dimensional numerical arrays (tabular arrays are not yet supported).

A source-only alpha release can be found on the GitHub `releases <https://github.com/asdf-format/libasdf/releases>`__ page, with a binary release on the conda-forge package repository coing soon. An upcoming release will also include basic write support for ASDF files.

Build instructions, usage, and API documentation can be found in the official `libasdf documentation <https://libasdf.readthedocs.io/en/latest/>`__.
